Елемент article

Google Chrome

Google Chrome - це веб-браузер, розроблений Google, випущений в 2008 році. Chrome - найбільш популярний веб-браузер в світі на сьогоднішній день!

Mozilla Firefox

Mozilla Firefox - це веб-браузер з відкритим початковим кодом, розроблений Mozilla. Firefox був другим по популярності веб-браузером з січня 2018 року.

Microsoft Edge

Microsoft Edge - це веб-браузер, розроблений Microsoft, випущений в 2015 році. Microsoft Edge замінив Internet Explorer.
